# 🎓 Student Performance Prediction & Analysis

## 📌 Overview
This project predicts student academic performance using Machine Learning techniques. It includes data preprocessing, visualization, model comparison, and a prediction system based on user inputs.

---

## 🚀 Features
- Data Cleaning & Preprocessing
- Data Visualization using Matplotlib & Seaborn
- Model Comparison:
  - Linear Regression
  - Random Forest Regressor
- Feature Importance Analysis
- Interactive Prediction System

---

## 📊 Results
- Linear Regression R² Score: ~0.97
- Random Forest R² Score: ~0.98

---

## 🔍 Key Insights
- Study hours and subject scores have the highest impact on performance
- Attendance significantly affects overall score
- Random Forest outperformed Linear Regression

---

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📈 Sample Prediction
Input:
- Study Hours: 5
- Attendance: 80%
- Subject Scores: 70–75 range

Output:
- Predicted Score: ~72–86
